📌 Overview

The Automata Simulator with Animation is an interactive web application that helps visualize and simulate core concepts of Theory of Computation. It provides animated, step-by-step execution of different automata, making complex concepts easier to understand.

---

🎯 Key Highlights

- 🎬 Real-time step-by-step animation
- 🧠 Interactive learning experience
- 📊 Visual state transition diagrams
- ⚡ Lightweight and browser-based (no installation required)

---

✨ Features

🔹 Supported Automata

- Deterministic Finite Automata (DFA)
- Non-Deterministic Finite Automata (NFA)
- ε-NFA
- Regular Expressions
- Pushdown Automata (PDA)

🔹 Functionalities

- Input binary strings (0s and 1s)
- Run full animation or step-by-step execution
- Reset simulation anytime
- Download automata diagrams
- Clean and responsive UI
